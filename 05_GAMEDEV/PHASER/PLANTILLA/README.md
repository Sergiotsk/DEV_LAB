# Proyecto Phaser

Proyecto base creado a partir de la plantilla.

## Cómo ejecutar

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Iniciar servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Abrir en el navegador la URL que aparece (usualmente `http://localhost:5173`)

## Estructura

- `src/main.js`: Código principal del juego
- `index.html`: Punto de entrada
- `style.css`: Estilos globales
